# Express TypeScript Project

A basic Express.js project with TypeScript setup.

## Prerequisites

- Node.js
- pnpm

## Installation

```bash
pnpm install
```

## Development

To start the development server with hot-reload:

```bash
pnpm dev
```

## Building for Production

To build the project:

```bash
pnpm build
```

To start the production server:

```bash
pnpm start
```
